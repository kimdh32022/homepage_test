연결 선택자

1

인접 형제 선택자 : 같은 부모를 가진 형제 요소 중 첫번째 동생 요소에만 스타일을 적용

연결 선택자란 선택자와 선택자를 연결해 적용 대상을 제한하는 선택자.

컴비네이션 선택자(combination selector) 또는 '조합선택자'라고 함.

주의점 : 연결 선택자는 캐스케이딩때문에 자식 요소(하위의 요소)와 손자 요소(하위의 하위 요소)를 다 변환시킴

형제 선택자

2

형제 선택자란 형제 요소들에 스타일을 적용하는 방식.
인접 형제 선택자와 다른 점은 모든 형제 요소에 다 적용 됨.
속성 선택자

3

속성 선택자란 지정한 속성을 가진 요소를 찾아 스타일 적용하는 것.

  • 햄버거
  • 피자
    • [속성 = 값] 주어진 속성과 속성 값이 일치하는 요소를 찾아 스타일 적용
  • 치킨
    • [속성 ~= 값]여러 속성 값 중에 해당 값이 포함되어 있는 요소를 찾아 스타일 적용
  • 족발
    • [속성 ~= 값]여러 속성 값 중에 해당 값이 포함되어 있는 요소를 찾아 스타일 적용
